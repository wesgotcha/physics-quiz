# 大学物理C 选择题题库

在线刷题系统，覆盖大学物理C课程全部章节的选择题，支持 LaTeX 公式渲染和题目配图。

**线上地址：** `https://wesgotcha.github.io/physics-quiz/`

## 本地运行

```bash
npm start
```

浏览器打开 `http://localhost:3000/` 进入刷题页面。

## 刷题页面 (index.html)

- **选题：** 按章节展开，勾选题目后点击"开始测验"
- **答题：** 点击 A/B/C/D 选项，自动判断对错，0.85 秒后自动跳下一题
- **配图：** 有配图的题目会显示图片，默认半宽显示，**点击图片可放大/缩小**
- **错题本：** 答错的题自动记录，可点击"只刷错题"针对性复习
- **成绩单：** 完成后显示正确率和题号网格，点击题号可跳回复查

## 编辑模式 (admin.html) — 本地专属

> ⚠️ 编辑模式需要本地 Node.js 服务器，**不部署到 GitHub Pages**。

### 启动

```bash
npm start
# 访问 http://localhost:3000/admin.html
```

### 编辑题目

1. 左侧章节树点击题目 → 右侧加载编辑表单
2. 修改题干/选项（支持 LaTeX 公式，实时预览渲染效果）
3. 选择正确答案
4. `Ctrl+S` 或点击"保存"
5. 修改立即生效，刷新刷题页即可看到

### 上传配图

1. 在编辑器中定位到需要配图的题目
2. 点击"选择文件" → 选择图片（支持 PNG/JPEG/GIF/SVG/WebP，≤5MB）
3. 文件名自动生成为 `{章节号}-q{题号}.{扩展名}`，无需手动命名
4. 点击"上传" → 图片保存到服务器
5. 点击"保存" → 题目与图片关联

### 更换/删除配图

- **更换：** 选择新文件上传即可覆盖
- **删除：** 点击"删除图片"再保存

### 原始 JSON 编辑器

点击"切换到原始JSON编辑器"可编辑完整数据库 JSON，适合批量操作。编辑完后保存会自动同步到刷题页面。

### 图片命名规则

图片存储在 `public/images/`，命名格式：

```
{章节号}-q{题号}.{扩展名}

例如：4-q12.png  → 第4章第12题的配图
      7-q3.png   → 第7章第3题的配图
```

文件名由编辑器自动生成，不需要手动输入。

## 部署到 GitHub Pages

```bash
# 1. 构建部署文件到 public/ 目录
npm run build:public

# 2. 将 public/ 推送到 gh-pages 分支
git subtree push --prefix public origin gh-pages
```

> `build:public` 会自动排除 admin.html（编辑模式不上线）。

## 项目结构

```
physics-quiz/
├── index.html          # 刷题页面
├── admin.html          # 编辑模式（本地专属）
├── server.js           # Node.js 本地服务器
├── data/
│   └── physics_quiz_db.json  # 题库数据
├── public/             # 部署目录（GitHub Pages 源）
│   ├── index.html
│   ├── data/
│   └── images/         # 题目配图（70张）
├── scripts/            # 构建/审计脚本
└── tmp/                # 临时文件（OCR 中间产物、来源 PDF）
```

## 技术栈

- 纯 HTML/CSS/JS，无前端框架
- MathJax 3 渲染 LaTeX 公式
- Node.js 原生 http 模块后端
- JSON 文件存储题库数据
